## Installation

1. Create a new file called `ten.tsx` inside the `components/animata/bento-grid` directory.

   ```bash
   mkdir -p components/animata/bento-grid && touch components/animata/bento-grid/ten.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/bento-grid/ten.tsx

```
