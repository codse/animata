# Installation

1. Create a new file called `five.tsx` inside the `components/animata/bento-grid` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/bento-grid && touch components/animata/bento-grid/five.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\bento-grid && type null > components\animata\bento-grid\five.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/bento-grid/five.tsx

```
