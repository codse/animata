# Installation

1. Create a new file called `spinner.tsx` inside the `components/animata/progress` directory.

   macOS/Linux:
   ```bash
   mkdir -p components/animata/progress && touch components/animata/progress/spinner.tsx
   ```

   windows:
   ```bash
   mkdir components\animata\progress && type null > components\animata\progress\spinner.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/progress/spinner.tsx

```
