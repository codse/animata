# Installation

1. Create a new file called `shift-tabs.tsx` inside the `components/animata/container` directory.

   macOS/Linux:
   ```bash
   mkdir -p components/animata/container && touch components/animata/container/shift-tabs.tsx
   ```

   windows:
   ```bash
   mkdir components\animata\container && type null > components\animata\container\shift-tabs.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/container/shift-tabs.tsx

```
