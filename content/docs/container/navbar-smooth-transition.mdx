## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion lucide-react
```

<Step>Update `tailwind.config.js`</Step>

Add the following to your tailwind.config.js file.

```json
module.exports = {
  theme: {
    extend: {
    }
  }
}
```

<Step>Run the following command</Step>

It will create a new file `navbar-smooth-transition.tsx` inside the `components/animata/navbar` directory.

```bash
mkdir -p components/animata/navbar && touch components/animata/navbar/navbar-smooth-transition.tsx
```
