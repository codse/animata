## Installation

1. Create a new file called `nav-tabs.tsx` inside the `components/animata/container` directory.

   ```bash
   mkdir -p components/animata/container && touch components/animata/container/nav-tabs.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/container/nav-tabs.tsx

```
