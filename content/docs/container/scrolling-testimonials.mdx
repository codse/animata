# Installation

1. Create a new file called `scrolling-testimonials.tsx` inside the `components/animata/container` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/container && touch components/animata/container/scrolling-testimonials.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\container && type null > components\animata\container\scrolling-testimonials.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/container/scrolling-testimonials.tsx

   ```
