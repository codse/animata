## Installation

1. Create a new file called `carousel.tsx` inside the `components/animata/container` directory.

   ```bash
   mkdir -p components/animata/container && touch components/animata/container/carousel.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/container/carousel.tsx

   ```
