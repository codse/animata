### Manual

<Steps>

<Step>Install dependencies</Step>

This component uses the [Marquee](/docs/container/marquee) component for the scrolling track. Make sure it's in your project first.

<Step>Run the following command</Step>

```bash
mkdir -p components/animata/container && touch components/animata/container/announcement-ribbon.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/container/announcement-ribbon.tsx

```
