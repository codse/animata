# Installation

1. Create a new file called `apk-splash-screen.tsx` inside the `components/animata/splash-screen` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/splash-screen && touch components/animata/splash-screen/apk-splash-screen.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\splash-screen && type null > components\animata\splash-screen\apk-splash-screen.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/splash-screen/apk-splash-screen.tsx

   ```
