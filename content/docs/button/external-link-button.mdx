## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install @radix-ui/react-icons
```

<Step>Run the following command</Step>

It will create a new file called `external-link-button.tsx` inside the `components/animata/button` directory.

```bash
mkdir -p components/animata/button && touch components/animata/button/external-link-button.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/button/external-link-button.tsx

```

</Steps>
