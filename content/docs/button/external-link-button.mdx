## Installation

1. Create a new file called `external-link-button.tsx` inside the `components/animata/button` directory.

   ```bash
   mkdir -p components/animata/button && touch components/animata/button/external-link-button.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/button/external-link-button.tsx

   ```
