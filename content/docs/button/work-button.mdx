# Installation

1. Create a new file called `work-button.tsx` inside the `components/animata/button` directory.

   macOS/Linux:
   ```bash
   mkdir -p components/animata/button && touch components/animata/button/work-button.tsx
   ```

   windows:
   ```bash
   mkdir components\animata\button && type null > components\animata\button\work-button.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/button/work-button.tsx

   ```
