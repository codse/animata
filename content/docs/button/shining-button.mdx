# Installation

1. Create a new file called `shining-button.tsx` inside the `components/animata/button` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/button && touch components/animata/button/shining-button.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\button && type null > components\animata\button\shining-button.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/button/shining-button.tsx

   ```
