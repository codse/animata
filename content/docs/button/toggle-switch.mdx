# Installation

1. Create a new file called `toggle-switch.tsx` inside the `components/animata/button` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/button && touch components/animata/button/toggle-switch.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\button && type null > components\animata\button\toggle-switch.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/button/toggle-switch.tsx

   ```
