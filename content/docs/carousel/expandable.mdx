# Installation

1. Install dependencies.

   ```bash
   yarn add framer-motion
   ```

2. Create a new file called `expandable.tsx` inside the `components/animata/carousel` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/carousel && touch components/animata/carousel/expandable.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\carousel && type null > components\animata\carousel\expandable.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/carousel/expandable.tsx

   ```
