# Installation

1. Create a new file called `skew-image.tsx` inside the `components/animata/image` directory.

macOS/Linux:

```bash
mkdir -p components/animata/image && touch components/animata/image/skew-image.tsx
```

windows:

```bash
mkdir components\animata\image && type null > components\animata\image\skew-image.tsx
```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/image/skew-image.tsx

```
