# Installation

1. Create a new file called `tilted-cover.tsx` inside the `components/animata/image` directory.

macOS/Linux:

```bash
mkdir -p components/animata/image && touch components/animata/image/tilted-cover.tsx
```

windows:

```bash
mkdir components\animata\image && type null > components\animata\image\tilted-cover.tsx
```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/image/tilted-cover.tsx

```
