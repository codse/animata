# Installation

1. Create a new file called `shape-shifter.tsx` inside the `components/animata/hero` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/hero && touch components/animata/hero/shape-shifter.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\hero && type null > components\animata\hero\shape-shifter.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/hero/shape-shifter.tsx

   ```
