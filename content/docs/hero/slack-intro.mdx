## Installation

1. Create a new file called `slack-intro.tsx` inside the `components/animata/hero` directory.

   ```bash
   mkdir -p components/animata/hero && touch components/animata/hero/slack-intro.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/hero/slack-intro.tsx

   ```
