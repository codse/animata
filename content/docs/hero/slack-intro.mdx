## Installation

<Steps>

<Step>Install dependencies</Step>

This uses [WaveReveal](/docs/text/wave-reveal) for the text. Install it by following the instructions [here](/docs/text/wave-reveal#installation). You can use simple text only if you don't want to use WaveReveal.

<Step>Run the following command</Step>

It will create a new file called `slack-intro.tsx` inside the `components/animata/hero` directory.

```bash
mkdir -p components/animata/hero && touch components/animata/hero/slack-intro.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/hero/slack-intro.tsx

```

</Steps>
