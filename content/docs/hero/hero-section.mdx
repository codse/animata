## Installation

### CLI

<RegistryInstall category="hero" name="hero-section" />

### Manual

<Steps>

<Step>Add to your CSS</Step>
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
@theme {
  --animate-fade-in: fadeIn 0.5s ease-in;
}
```

<Step>Run the following command</Step>

It will create a new file called `hero-section.tsx` inside the `components/animata/hero` directory.

```bash
touch components/animata/hero/hero-section.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/hero/hero-section.tsx

```

</Steps>
