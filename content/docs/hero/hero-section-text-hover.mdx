# Installation

1. Install dependencies.
   ```bash
   yarn add lucide-react
   ```
2. Create a new file called `hero-section-text-hover.tsx` inside the `components/animata/hero` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/hero && touch components/animata/hero/hero-section-text-hover.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\hero && type null > components\animata\hero\hero-section-text-hover.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/hero/hero-section-text-hover.tsx

   ```
