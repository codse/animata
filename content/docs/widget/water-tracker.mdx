## Installation

1. Install dependencies

```bash
npm install lucide-react
```

2. Create a new file called `water-tracker.tsx` inside the `components/animata/widget` directory.

   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/water-tracker.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/widget/water-tracker.tsx

   ```
