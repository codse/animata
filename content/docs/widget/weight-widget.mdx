## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install lucide-react
```

<Step>Run the following command</Step>

It will create a new file called `weight-widget.tsx` inside the `components/animata/widget` directory.

```bash
mkdir -p components/animata/widget && touch components/animata/widget/weight-widget.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/widget/weight-widget.tsx

```

</Steps>
