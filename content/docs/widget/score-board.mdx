## Installation

1. Create a new file called `score-board.tsx` inside the `components/animata/widget` directory.

   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/score-board.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/widget/score-board.tsx

```
