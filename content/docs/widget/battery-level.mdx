# Installation

1. Create a new file called `battery-level.tsx` inside the `components/animata/widget` directory.

   macOS/Linux:
   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/battery-level.tsx
   ```

   windows:
   ```bash
   mkdir components\animata\widget && type null > components\animata\widget\battery-level.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/widget/battery-level.tsx

```
