## Installation

1. Create a new file called `calorie-counter.tsx` inside the `components/animata/widget` directory.

   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/calorie-counter.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/widget/calorie-counter.tsx

   ```
