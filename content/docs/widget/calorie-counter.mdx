## Installation

<Steps>

<Step>Install dependencies</Step>

This uses [DonutChart](/docs/graphs/donut-chart) for the progress bar. Install it by following the instructions [here](/docs/graphs/donut-chart#installation).

<Step>Run the following command</Step>

It will create a new file called `calorie-counter.tsx` inside the `components/animata/widget` directory.

```bash
mkdir -p components/animata/widget && touch components/animata/widget/calorie-counter.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/widget/calorie-counter.tsx

```

</Steps>
