## Installation

1. Create a new file called `weather-card.tsx` inside the `components/animata/widget` directory.

   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/weather-card.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/widget/weather-card.tsx

```
