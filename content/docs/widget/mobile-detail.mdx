# Installation

1. Create a new file called `mobile-detail.tsx` inside the `components/animata/widget` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/mobile-detail.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\widget && type null > components\animata\widget\mobile-detail.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/widget/mobile-detail.tsx

```
