# Installation

1. Install dependencies

```bash
yarn add lucide-react
```

2. Create a new file called `alarm-clock.tsx` inside the `components/animata/widget` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/widget && touch components/animata/widget/alarm-clock.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\widget && type null > components\animata\widget\alarm-clock.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/widget/alarm-clock.tsx

   ```
