## Installation

1. Create a new file called `grid.tsx` inside the `components/animata/background` directory.

   ```bash
   mkdir -p components/animata/background && touch components/animata/background/grid.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/background/grid.tsx

   ```
