# Installation

1. Create a new file called `diagonal-lines.tsx` inside the `components/animata/background` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/background && touch components/animata/background/diagonal-lines.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\background && type null > components\animata\background\diagonal-lines.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/background/diagonal-lines.tsx

   ```
