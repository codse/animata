# Installation

1. Create a new file called `moving-gradient.tsx` inside the `components/animata/background` directory.

macOS/Linux:

```bash
mkdir -p components/animata/background && touch components/animata/background/moving-gradient.tsx
```

windows:

```bash
mkdir components\animata\background && type null > components\animata\background\moving-gradient.tsx
```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/background/moving-gradient.tsx

```
