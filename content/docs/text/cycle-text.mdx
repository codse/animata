# Installation

1. Create a new file called `cycle-text.tsx` inside the `components/animata/text` directory.

   macOS/Linux:
   ```bash
   mkdir -p components/animata/text && touch components/animata/text/cycle-text.tsx
   ```

   windows:
   ```bash
   mkdir components\animata\text && type null > components\animata\text\cycle-text.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/text/cycle-text.tsx

   ```
