## Installation

### CLI

<RegistryInstall category="text" name="bottom-up-letters" />

### Manual

<Steps>

<Step>Install the shared runtime</Step>

Every preset in the text category shares its WAAPI loop + scoped stage
styles with the same `text-animator` runtime. Skip this step if you
already added it for another text preset.

```bash
mkdir -p components/animata/text && touch components/animata/text/text-animator.tsx components/animata/text/text-animator.css
```

Paste the runtime into `text-animator.tsx`:

```tsx file=<rootDir>/animata/text/text-animator.tsx

```

And the styles into `text-animator.css`:

```css file=<rootDir>/animata/text/text-animator.css

```

<Step>Run the following command</Step>

It will create a new file called `bottom-up-letters.tsx` inside the `components/animata/text` directory.

```bash
mkdir -p components/animata/text && touch components/animata/text/bottom-up-letters.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/text/bottom-up-letters.tsx

```

</Steps>
