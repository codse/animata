# Installation

1. Create a new file called `bold-copy.tsx` inside the `components/animata/text` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/text && touch components/animata/text/bold-copy.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\text && type null > components\animata\text\bold-copy.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/text/bold-copy.tsx

   ```
