# Installation

1. Install dependencies

   ```bash
   yarn add framer-motion lucide-react
   ```

2. Create a new file called `modal.tsx` inside the `components/animata/overlay` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/overlay && touch components/animata/overlay/modal.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\overlay && type null > components\animata\overlay\modal.tsx
   ```

3. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/overlay/modal.tsx

```
