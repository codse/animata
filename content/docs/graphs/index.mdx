<Callout>
This is not a full-fledged chart library. It's a collection of simple graphs that can be used in feature presentations or widgets.

For more complex use cases, consider using libraries like [Recharts](https://recharts.org/en-US/) or [Chart.js](https://www.chartjs.org/).

</Callout>

Currently, the following graphs are available:

1. [Bar Chart](/docs/graphs/bar-chart)
2. [Commit Graph](/docs/graphs/commit-graph)
3. [Donut Chart](/docs/graphs/donut-chart)
4. [Gauge Chart](/docs/graphs/gauge-chart)
5. [Progress](/docs/graphs/progress)
6. [Ring Chart](/docs/graphs/ring-chart)
