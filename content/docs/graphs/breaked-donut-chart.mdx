# Installation

1. Create a new file called `breaked-donut-chart.tsx` inside the `components/animata/graphs` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/graphs && touch components/animata/graphs/breaked-donut-chart.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\graphs && type null > components\animata\graphs\breaked-donut-chart.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/graphs/breaked-donut-chart.tsx

   ```
