## Installation

1. Create a new file called `ring-chart.tsx` inside the `components/animata/graphs` directory.

   ```bash
   mkdir -p components/animata/graphs && touch components/animata/graphs/ring-chart.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/graphs/ring-chart.tsx

```
