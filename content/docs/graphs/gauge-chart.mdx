## Installation

<Steps>

<Step>Update `tailwind.config.js`</Step>
```js
theme: {
    extend: {
     colors: {
        foreground: "hsl(var(--foreground))",
     }
    },
  },
```

<Step>Run the following command</Step>

It will create a new file called `gauge-chart.tsx` inside the `components/animata/graphs` directory.

```bash
mkdir -p components/animata/graphs && touch components/animata/graphs/gauge-chart.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/graphs/gauge-chart.tsx

```

</Steps>
