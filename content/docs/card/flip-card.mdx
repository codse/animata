# Installation

1. Create a new file called `flip-card.tsx` inside the `components/animata/card` directory.

   macOS/Linux:
   ```bash
   mkdir -p components/animata/card && touch components/animata/card/flip-card.tsx
   ```

   windows:
   ```bash
   mkdir components\animata\card && type null > components\animata\card\flip-card.tsx
   ```

2. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/card/flip-card.tsx

```
