## Installation

1. Install dependencies

   ```bash
   npm install framer-motion
   ```

2. Create a new file called `flipping-card.tsx` inside the `components/animata/card` directory.

   ```bash
   mkdir -p components/animata/card && touch components/animata/card/flipping-card.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/card/flipping-card.tsx

   ```
