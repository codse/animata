# Installation

1. Install dependencies

```bash
yarn add lucide-react
```

2. Create a new file called `swap-card.tsx` inside the `components/animata/card` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/card && touch components/animata/card/swap-card.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\card && type null > components\animata\card\swap-card.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/card/swap-card.tsx

   ```
