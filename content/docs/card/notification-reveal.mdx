## Installation

<Steps>
<Step>Install dependencies</Step>

```bash
npm install framer-motion
```

<Step>Run the following command</Step>

It will create a new file called `notification-reveal.tsx` inside the `components/animata/card` directory.

```bash
mkdir -p components/animata/card && touch components/animata/card/notification-reveal.tsx
```

<Step>Paste the code</Step>

Open the newly created file and paste the following code:

```jsx file=<rootDir>/animata/card/notification-reveal.tsx

```

</Steps>
