# Installation

1. Create a new file called `animated-card.tsx` inside the `components/animata/card` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/card && touch components/animata/card/animated-card.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\card && type null > components\animata\card\animated-card.tsx
   ```
