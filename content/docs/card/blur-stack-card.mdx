## Installation

1. Install dependencies.

   ```bash
   npm install framer-motion
   ```

2. Create a new file called `blur-stack-card.tsx` inside the `components/animata/card` directory.

   ```bash
   mkdir -p components/animata/card && touch components/animata/card/blur-stack-card.tsx
   ```

3. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/card/blur-stack-card.tsx

   ```
