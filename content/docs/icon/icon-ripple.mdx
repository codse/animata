# Installation


1. Install dependencies.

   ```bash
   yarn add lucide-react
   ```

2. Create a new file called `icon-ripple.tsx` inside the `components/animata/icon` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/icon && touch components/animata/icon/icon-ripple.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\icon && type null > components\animata\icon\icon-ripple.tsx
   ```


3. Open the newly created file and add the following code:

```jsx file=<rootDir>/animata/icon/icon-ripple.tsx

```
