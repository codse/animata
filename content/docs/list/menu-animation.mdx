## Installation

1. Create a new file called `menu-animation.tsx` inside the `components/animata/list` directory.

   ```bash
   mkdir -p components/animata/list && touch components/animata/list/menu-animation.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/list/menu-animation.tsx

   ```
