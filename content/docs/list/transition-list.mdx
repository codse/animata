# Installation

1. Create a new file called `transition-list.tsx` inside the `components/animata/list` directory.

   macOS/Linux:

   ```bash
   mkdir -p components/animata/list && touch components/animata/list/transition-list.tsx
   ```

   windows:

   ```bash
   mkdir components\animata\list && type null > components\animata\list\transition-list.tsx
   ```

2. Open the newly created file and add the following code:

   ```jsx file=<rootDir>/animata/list/transition-list.tsx

   ```
