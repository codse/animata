New components, bug fixes, and infrastructure changes, going back to the beta launch in June 2024.

## Recent releases

### [April 2026](/docs/changelog/2026-04)

19 new text animation presets built on a shared WAAPI runtime. shadcn registry went live — every component installable with `npx shadcn add`. Announcement ribbon added, site revamped for speed and accessibility.

### [March 2026](/docs/changelog/2026-03)

Stack upgrade: Next.js 16, Tailwind CSS v4, Storybook 10, and Biome replacing the old linter setup.
